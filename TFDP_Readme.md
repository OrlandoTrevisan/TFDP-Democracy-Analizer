### Project Name

TFDP Democracy Analyzer

### Overview

The TFDP Democracy Analyzer is a Python application developed as part of the Code in Place 2026 Final Project.

TFDP stands for The Future of Democracy Project, an educational initiative dedicated to helping students and citizens better understand democracy, democratic institutions, and the challenges democracies face in the modern world.

The application evaluates countries using five democratic indicators:

* Electoral Integrity
* Civil Liberties
* Government Transparency
* Education Index
* Economic Stability

Based on these indicators, the program calculates a weighted TFDP Score and classifies countries into democratic categories.

### Features

* Create new democracy analyses
* Store analyses in a JSON database
* View analysis history
* Compare countries
* Display democracy dashboards
* Generate world rankings
* Calculate database statistics
* Produce democratic assessments
