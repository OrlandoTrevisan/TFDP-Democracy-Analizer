# Project Description

The TFDP Democracy Analyzer was created as part of the Code in Place 2026 Final Project.

The project is inspired by a broader initiative called The Future of Democracy Project (TFDP), which seeks to explore how democratic systems evolve and how citizens can better understand democratic institutions.

The application allows users to evaluate a country's democratic quality using five key indicators:

* Electoral Integrity
* Civil Liberties
* Government Transparency
* Education Index
* Economic Stability

Each indicator receives a score between 0 and 100. The program calculates a weighted TFDP Score and classifies the country into one of five categories:

* Strong Democracy
* Stable Democracy
* Democracy at Risk
* Fragile Democracy
* Emerging Authoritarianism

The program also stores analyses in a JSON database, allowing users to compare countries, generate rankings, visualize democratic indicators through dashboards, and calculate overall database statistics.

I chose this project because democracy is one of the most important political and social topics in the world today. My goal was to create a simple educational tool that could eventually evolve into a larger platform for democratic analysis and civic education.

The current version uses manually entered data, but future versions will integrate real-world datasets from international organizations and academic institutions to provide automated democratic assessments.
