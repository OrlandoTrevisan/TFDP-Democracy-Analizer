Visão geral
O TFDP Democracy Analyzer é um aplicativo em Python desenvolvido como parte do Projeto Final Code in Place 2026.

TFDP significa The Future of Democracy Project (Projeto Futuro da Democracia), uma iniciativa educacional dedicada a ajudar estudantes e cidadãos a compreender melhor a democracia, as instituições democráticas e os desafios que as democracias enfrentam no mundo moderno.

O aplicativo avalia os países usando cinco indicadores democráticos:

Integridade Eleitoral
Liberdades Civis
Transparência governamental
Índice de Educação
Estabilidade Econômica
Com base nesses indicadores, o programa calcula uma pontuação TFDP ponderada e classifica os países em categorias democráticas.

Características
Criar novas análises sobre democracia
Armazene as análises em um banco de dados JSON.
Ver histórico de análises
Comparar países
Exibir painéis de democracia

Calcular estatísticas do banco de dados
Produzir avaliações democráticas

Desenvolvimento futuro
Esta versão é um protótipo.

As versões futuras incorporarão fontes de dados do mundo real para calcular automaticamente indicadores democráticos e monitorar tendências democráticas ao longo do tempo.

O objetivo a longo prazo é apoiar a pesquisa, a educação e a compreensão pública da democracia em todo o mundo.
